# Zoom Morsels 

An openFrameworks program that tracks multiple faces from multiple camera streams. There are a number of interaction techniques that allow for each individual to control their camera view. Read more about this project on [the original post](https://courses.ideate.cmu.edu/60-461/s2020/cbromsandrew-cmu-edu/04/30/christian-final/).
